什么是 OpenClaw？

本部分将带您了解 OpenClaw 的本质以及它为何需要特定的硬件支持。了解这些基础，有助于您后续顺利运行这个“数字员工”。

OpenClaw 本质上是一个“AI 数字员工” (AI Agent)。它的大脑是 AI 大模型（如 Minimax、DeepSeek），而它的身体是你的电脑或服务器。它能帮你自动上网、操作软件、回复消息、处理数据。

🖥 电脑配置要求

  • ✔ Windows 电脑： 推荐 16G 内存。
  • ✔ 苹果 Mac： 推荐 M 系列芯片（10核），16G 内存。

为什么一个聊天机器人需要这么高的配置？

因为它不仅是聊天。当它执行复杂任务（抓取数据、处理大型 Excel、运行小型识别模型）时，需要同时开启多个后台浏览器和进程，极其消耗内存。

部署方式大比拼

本部分通过多维度数据对比，帮助新手小白在“本地电脑”和“云端服务器”之间做出最合适的选择。

本地部署 vs 云服务器

* 分数越高代表该项体验越好/优势越大

💻 本地部署 (装自己电脑)

  • 资金成本： 0元 (利用现有电脑)
  • 网络要求： 较高 (有时需网络辅助连接GitHub)
  • 在线时间： 电脑关机，AI即下班
  • 技术门槛： 极低 (适合小白尝鲜)
  • 数据隐私： 绝对安全 (文件在本地硬盘)

☁ 云端部署 (租用服务器)

  • 资金成本： 约 30~100元/月
  • 网络要求： 极其稳定 (推荐海外轻量云)
  • 在线时间： 7×24小时在线
  • 技术门槛： 中等 (需使用 1Panel 等面板)
  • 数据隐私： 相对安全 (存于云端)

👑 最终建议路径

第一阶段：白嫖试水

先在本地电脑装！用免费 Token 试试看，确认它能干活了再考虑下一步。

第二阶段：正式上岗

觉得好用，需要半夜回消息、自动发帖，再花几十块租云服务器挂在云端。

保姆级安装与使用

本部分以“本地部署试水”为主线，带您零基础一步步将 OpenClaw 运行起来。请点击下方步骤查看详情。

步骤 1：准备“大脑”

OpenClaw 只是躯干，你需要给它接入大脑才能进行思考。

A. 去 Minimax 官网 (agent.minimaxi.com) 或 DeepSeek 官网注册账号。
B. 找到“API 开放平台”或“开发者选项”。
C. 生成一个 API Key（一串很长的英文字母和数字），这就是你的“Token”凭证，复制并保存好。

步骤 2：请“军师”帮忙

利用现有的 AI 工具来获取最新的安装脚本，避免踩坑。

在 Minimax 或 Kimi 对话框输入：

"我是零基础小白，使用的是 Windows (或 Mac) 电脑，请手把手教我如何使用一键脚本安装 OpenClaw，请给出最简单的步骤。"

➡ AI 会返回给你一段一键安装的代码（通常是一段以 curl 开头的命令）。

步骤 3：本地执行安装

在您的电脑上运行刚才获取的指令。

  • 打开终端：
    Windows：按下 Win + R，输入 cmd 回车。
    Mac：打开“终端 (Terminal)”应用。
  • 粘贴代码：
    将 AI 给你的那段代码复制进去，按回车。
  • 耐心等待：
    喝杯茶等待它自动下载和安装（若卡住报错大概率是网络问题，可向AI索要国内镜像方案）。

步骤 4：填入 Token

激活您的数字员工并开始使用。

🃏
1. 打开网址
安装完成后，终端会提示网址（如 localhost:18789），在浏览器打开。
⚙
2. 进入设置
在 Settings 中找到填模型和 API Key 的地方。
🔑
3. 填入凭证
将步骤1申请的 API Key 填入并保存。
🎉
4. 大功告成
现在可以在界面里给它安排工作了！
